E-wish

E-wish is a project to send wishes to family and loved ones. No Registration required!! Just add the fields and a link is made, ready to be sent to the recipients. Currently it is built in Flask (for Backend) and some HTML, CSS (for Frontend). Later on there will be more added into it.

Thanks to @ammarsaa (the name is a link) for the UI code from UIVERCE. It was super helpful.

This is a project for Arcade by George Vazakas.
